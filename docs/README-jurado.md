# Material complementario – Tesis Doctoral

Este repositorio contiene el material complementario para la evaluación de la tesis.

## Contenido

- DSL textual: carpeta `dsl/`
- Metamodelos: carpeta `metamodelos/`
- Transformaciones M2M/M2T: carpeta `transformaciones/`
- Ejemplos derivados: carpeta `ejemplos/`

Los metamodelos se incluyen en formato editable y de alta calidad para una mejor visualización que en el PDF.
